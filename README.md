## Deskripsi Proyek

Aplikasi ini dibuat menggunakan Flutter untuk perangkat Android sebagai tempat bagi para penggemar anime menemukan, mengikuti, dan berdiskusi tentang anime favorit mereka. Berkat Firebase, aplikasi ini menyediakan login yang mudah, penyimpanan data yang cepat, serta fitur sosial yang bikin pengguna bisa terhubung satu sama lain.
